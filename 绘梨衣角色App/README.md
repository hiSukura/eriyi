# 绘梨衣 — 角色档案 App

龙族（Dragon Raja）角色 **上杉绘梨衣（Uesugi Erii）** 的移动端角色档案设计。

## 设计概览

- **工具**: Ardot (腾讯设计工具)
- **目标设备**: iPhone 14 (390 × 844 px)
- **设计风格**: Modern Dark Cinema Mobile
- **色彩系统**: 深海军蓝背景 (#0F0F23) × 金色点缀 (#CA8A04)
- **字体**: Noto Serif JP（标题） + Noto Sans JP（正文）

## 页面

| 页面 | 说明 |
|------|------|
| `01-角色档案.png` | 角色头像、身份信息、语录、标签胶囊、底部导航 |
| `02-物語剧情.png` | 三张时间线事件卡（初遇·东京都 → 小怪兽 → 最后的樱花） |
| `03-言灵能力.png` | 审判（Judgment）言灵详情、序列第111位、效果描述 |
