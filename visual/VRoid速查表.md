# 绘梨衣 VRoid 调参表

> 打开 VRoid Studio → New → Female 模板 → 照下面调 → Export VRM

## 脸 (Face)
| 位置 | 参数 | 值 |
|------|------|-----|
| Face Shape | Round/Soft | 0.75 |
| Chin | Pointed | 0.38 |
| Jaw | Narrow | 0.35 |
| Cheeks | Full | 0.60 |

## 眼 (Eyes)
| 参数 | 值 | 说明 |
|------|-----|------|
| Size | 0.65 | 偏大圆眼 |
| Outer Corner | Drooping 0.55 | 温柔垂眼 |
| Iris Color | **#CC7848** | 琥珀金棕 |
| Highlight | 2个 | 都要开 |
| Pupil | 0.50 | 正常 |

## 眉 (Eyebrows)
| 参数 | 值 |
|------|-----|
| Shape | Soft Arch 0.60 |
| Thickness | 0.40 |
| Color | **#8B6B5B** |

## 发 (Hair) ⭐最重要
| 阶 | 位置 | 颜色 |
|----|------|------|
| 1 | 发根 0% | `#FCE4D8` |
| 2 | 15% | `#F5C4B0` |
| 3 | 30% | `#E89878` |
| 4 | 50% | `#D86850` |
| 5 | 70% | `#C04038` |
| 6 | 发尾 100% | `#A03030` |

Length → Long 1.0 | Volume 0.70 | Tips Slight Curl 0.25
Parting → Center | Bangs → Swept Side

## 身体 (Body)
Height 0.40（娇小）| Shoulders Narrow 0.40 | Build Slim 0.55

## 皮肤 (Skin)
Base **#FDF0E8** | Shadow **#E8D5C0** | Blush 0.30

## 服装 (Outfit)
- 上: 白衬衫 #FAFAFA | 长袖 | 领口翻折
- 配: 红蝴蝶结 #E85840 | 颈前正中 | 大小 0.8
- 下: A字裙 #FAFAFA | 及膝 | 裙边红 #E85840
- 鞋: 乐福鞋 #4A2020
